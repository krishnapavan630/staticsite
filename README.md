# CineTolly — Tollywood Movie Booking App

A stunning, fully responsive movie-booking single-page app for the latest Tollywood
blockbusters. Built with plain **HTML, CSS & JavaScript** — no build step, no dependencies.

## Features
- **Auto-playing hero carousel** of trending movies with animated transitions
- **Now Showing** grid + **Coming Soon** horizontal rail
- **Full booking flow**: pick theatre → showtime → interactive seat map → payment summary → animated ticket
- **Custom animated gradient scrollbar** + a top scroll-progress bar
- **Reveal-on-scroll** animations (IntersectionObserver)
- **Mobile responsive** with a slide-in hamburger menu
- Poster images load from a CDN with **graceful gradient fallbacks** (looks great even offline)

## 📁 Files
| File | Purpose |
|------|---------|
| `index.html` | Markup / structure |
| `styles.css` | All styling, animations & responsiveness |
| `script.js`  | Movie data, carousel, booking logic |
